# x3mp3
Simple YouTube to MP3 converter with options for bitrate. 

### Bypass YouTube's bot detection
YouTube sometimes runs some checks to make sure it's a real person opening the video, and if they suspect it's a bot they force a login prompt.
You can bypass this by using an extension to get the cookies.txt file from youtube (while logged in) and place it in the project directory. -- It's best you don't use your main account and just make a dummy account if you do this!
